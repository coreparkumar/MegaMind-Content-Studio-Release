# Megamind Content Studio Release Repository

This folder is prepared as a publish-ready public release repository for Version 1.

## Contents

- Windows installer
- release notes
- English user instructions

## Publish Checklist

1. Create a new public GitHub repository
2. Copy this folder into that repository root
3. Push the repository
4. Create release `Version 1`
5. Upload the installer `.exe` and `.blockmap`

## Important Note

This installer is not a fully self-contained desktop release.

Users still need:

- Python installed
- Ollama installed
- the expected local Ollama model available
